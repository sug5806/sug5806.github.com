2019/09/10 카테고리 리눅스

개인적으로 공부한 내용을 정리하였습니다.

  • 0 : 표준 입력(<)
  • 1 : 표준 출력(>)
  • 2 : 표준 에러(2>)

cat 1> example.txt : cat 명령어로 표준입력을 받고 example.txt 파일이 없으면 만들고 있다면 기존 내용을 덮어씌워 표준 출력을 한다.

cat 1>> example.txt : cat 명령어로 표준입력을 받고 example.txt 파일이 없으면 만들고 있다면 기존 내용에 추가하여 표준 입력을 한다.

cat -k bla 2> error.txt : 발생한 오류를 error.txt 파일이 없으면 만들고 있다면 기존 내용을 덮어씌워 표준 에러를 출력한다.

cat -k bla 2>> error.txt : 발생한 오류를 error.txt 파일이 없으면 만들고 있다면 기존 내용에 추가하여 표준 에러를 출력한다.

cat < input.txt > hello.txt : cat 명령어로 input.txt의 파일에 있는 내용을 표준입력으로 받고 hello.txt에 표준출력을 한다.

date > date.txt : date 명령어의 결과를 date.txt에 표준 출력한다.

cut < date.txt --delimiter " " --fields 4 : 공백을 구분자로하여 4번째 열에 있는 것을 자른다.

A | B : A의 출력을 B의 입력으로 사용하여 실행

A | tee output.txt | B : A의 출력을 output.txt에 출력(쓰기)하고 B의 입력으로 사용한 다음 결과를 터미널에 출력한다

date | tee output.txt | B > output2.txt : A의 출력을 output.txt에 출력(쓰기)하고 B의 입력으로 사용한 다음 결과를 output2.txt에 출력(쓰기)한다. 터미널에서는 출력 X
