2019-04-05 HTML 기초태그

2019/04/05 카테고리 HTML

HTML 기초 태그

  • !DCTYPE html : 이 문서를 HTML5로 정의하겠다
  • html : html의 루트 요소
  • head : 문서에 대한 메타정보를 포함
  • title : 이 문서의 타이틀을 명시
  • body : 이 문서의 시각적인 페이지 컨텐츠 포함
  • h1 : heading을 정의
  • p : 단락

HTML의 태그는 <tagname>content</tagname> 으로 이루어져 있다

HTML Documents

  • 모든 HTML문서는 !DOCTYPE html로 타입정의를 하고 시작한다
  • HTML 문서 자체는 html로 시작하고 /html로 끝난다
  • HTML의 비주얼적인 부분은 body와 /body 사이에 작성한다

HTML Headings

  • HTML의 헤딩은 h1 ~ h6로 정의되어 있다
  • h1부터 h6까지는 글자 크기의 차이가 있다

HTML Links

  • HTML의 링크는 a태그로 정의되어 있다
  • href 속성에 주소를 적으면 된다.

HTML Images

  • html의 이미지는 img태그로 정의되어 있다
  • 원본파일(src), 택스트(alt), width와 height로 사진의 크기를 조절할 수 있다

HTML Buttons

  • HTML의 버튼은 button태그로 정의되어 있다

HTML Lists

  • HTML의 리스트는 ul(정렬되지 않은 리스트), ol(숫자로 정렬되는 리스트) 로 나뉘어 있다
