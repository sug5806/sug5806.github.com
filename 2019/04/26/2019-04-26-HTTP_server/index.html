HTTP Server

  1. 브라우저를 통해 사용자가 HTTP Request Message를 보내면

  2. 서버는 그 Message를 해석하고 Response Message로 응답한다

    1. 어느 페이지로 접속했느냐?
    2. Query String은 어떤 데이터를 가지고 있느냐?
    3. 특정 스크립트 요청이 있느냐?
    4. 최종 응답을 HTML, 다운로드는 파일로 할것이냐?

  3. HTTP Response만 하는 서버

    • 주소에 검색어값을 넘겨준다 -> 쿼리스패싱?
    • 쿼리스트링
    • ? : 쿼리스트링의 시작
    • & : 다음변수
