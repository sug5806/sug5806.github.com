장고 템플릿 문법 1

장고에서 화면을 렌더링 하기 위해서는 템플릿 엔진을 사용한다. 템플릿 엔진은 각자 자기만의 템플릿 문법을 제공하는데 여러가지 기본 문법과 태그 필터를 살펴본다

  1. 컨텍스트 변수 출력하기
    • 뷰에서 컨텍스트 변수로 값을 전달해준다면 해당 변수 이름을 가지고 출력할 수 있다
  1. 폼 출력하기
    Create, Update 뷰 같은 경우 폼을 출력한다. 폼 태그들을 출력할 때 p, li, table 등의 태그로 감싸진 형식으로 출력되게 만들 수 있다. 또 입력 폼에서는 csrf_token은 필수 이다.
