2019-05-17 Django에 Bootstrap 적용하기

2019/05/17 카테고리 Django

bootstrap 적용하기

  1. base.html에 bootstrap 설치하기

    • CDN 스크립트로 설치하기 : 리소스 절감효과, 최신버전을 자동 유지, CDN을 사용하면 접속 속도가 증가

    • 다운로드해서 설치하기 : 커스터마이징 가능, 도메인을 넘나들지 않아도 된다.(특정 위치에서는 더 빠를 수 있다)

    • jQuery Slim 버전은 ajax 기능이 빠져있다.

      • min 버전 : 용량을 최소화한 버전 -> 띄어쓰기 생략, 엔터 생략
      • beautify : min 버전은 소스코드를 알아보기 힘들기 때문에 beautify해서 다시 보기 좋게 만든다.

    • 모든 static 리소스들은 캐시가 걸린다. -> 수정사항 바로 반영되지 않음

      1. 웹 서버에서 캐시타임 조절 -> 자주 수정되는 경우 캐시 타임을 확 줄인다.(보통 1~2일 유지)
      2. 이미 받은것은 만료가 될때까지 갱신이 안됨 -> static 리소스의 파일명을 변경한다.(가장 쉬운 방법이지만 장고의 소스코드를 바꿔야하므로 귀찮은 방법)
      3. static 리소스 뒤에 Query String을 붙인다. -> 아예 캐싱이 안되도록 한다.(뒤에 Query string을 붙이면 다른 파일이라고 인식하여 다운받음)

    • HTML 파일에 적용하기

      • CSS 파일은 헤더 바로밑에
      • JS파일은 /body태그 바로위에 위치시킨다 - jQuery, popper, bootstrap 순으로 위치시킨다

기존에는 user-agent로 모바일, 데스크탑, 태블릿으로 구분을 했으나 현재에는 반응형 웹을통해 화면의 너비를 통해 기종 구분없이 화면이 달라진다

Grid system : Bootstrap에서 지원하는 내장 반응형웹

  • col-로 쪼갠 나머지를 다른 애들이 먹는다
  • 여백을 col로 지정만 해주고 내용 부분만 조절하면 된다.
