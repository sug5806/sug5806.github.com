2019-05-09 heroku에 장고 배포하기

2019/05/09 카테고리 Django

Heroku에 장고 배포하기

1. 모듈을 설치해야 한다.

  1. dj-database-url : 데이터베이스 관련 옵션을 변수로 쉽게 접근할 수 있게 해주는 유틸리티

  2. gunicorn : wsgi용 미들웨어 : 웹서버와 장고 사이에서 다리역할을 한다

  3. whitenoise : static 파일 서빙용 미들웨어

  4. psycopg2-binary : postgreSQL용 드라이버

2. pip freeze > requirements.txt 명령을 통해 현재 자신한테 설치되어 있는 모듈의 목록을 만든다

3. settings.py에서 세팅을 한다

4. git 설정

  • heroku login : 헤로쿠 계정으로 로그인한다
  • git init : 현재 폴더로 깃을 사용하겠다
  • git add -A . : 현재 변경 사항 전부를 add하겠다
  • git commit -m "message" : add한 내용을 commit을 하겠다

5. heroku app 만들기

  • heroku create [app_name] : app_name에서 특수문자는 - 만 사용가능하다

6. 업로드

  • git push heroku master

7. db 초기화

  • heroku run python manage.py migrate

8. 관리자 비밀번호 만들기

  • heroku run python manage.py createsuperuser

9. 확인하기

  • heroku open
